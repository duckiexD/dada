 ## Улучшение читаемости текста на фотографиях
Суть: Программа, которая делает текст на некачественных фото (блики, тень, размытие) более четким.

Базовые алгоритмы: Билатеральная фильтрация, адаптивная бинаризация (Оцу), морфологические операции.

Свой алгоритм: Комбинация нескольких методов с интеллектуальным выбором параметров на основе анализа изображения (например, оценка освещенности, обнаружение областей с текстом).

## Стек:
Python 3.8+
OpenCV 4.x (cv2) - основная библиотека компьютерного зрения
NumPy - работа с матрицами изображений
Matplotlib - визуализация результатов
Jupyter Notebook - интерактивная разработка и тестирование
