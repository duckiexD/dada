 ## Проект: Генетический алгоритм для игры в змейку

**Суть: Программа, которая анализирует игровую позицию, просчитывает возможные варианты на несколько ходов вперед и выбирает оптимальный ход, демонстрируя стратегическое поведение.**

**Базовые алгоритмы: Минимакс, Альфа-бета отсечение, оценочная функция на основе материального подсчета.**

**Свой алгоритм: Гибридная оценочная функция с позиционными весами и адаптивная стратегия поиска, изменяющая глубину анализа в зависимости от сложности позиции.**

**Стек технологий:**
Python 3.8+ - основной язык разработки генетического алгоритма и нейросетей
NumPy - эффективные матричные операции для нейронных сетей и вычисления приспособленности
PyGame - визуализация игрового процесса, отрисовка змейки и интерфейса обучения
Matplotlib - построение графиков эволюции показателей и анализа поколений
Jupyter Notebook - интерактивная разработка и тестирование компонентов алгоритма
