## Проект: Генетический алгоритм для обучения змейки

Суть: Система автоматического обучения AI-агента для игры в змейку с использованием эволюционных методов, где поколения агентов улучшаются через селекцию, скрещивание и мутацию.

Базовые алгоритмы: Генетический алгоритм, нейронные сети прямого распространения, отбор по приспособленности.

Свой алгоритм: Гибридный генетический алгоритм с прогрессивным обучением

Стек:
Python 3.8+ - основной язык разработки генетического алгоритма и нейросетей
NumPy - эффективные матричные операции для нейронных сетей и вычисления приспособленности
PyGame - визуализация игрового процесса, отрисовка змейки и интерфейса обучения
Matplotlib - построение графиков эволюции показателей и анализа поколений
Jupyter Notebook - интерактивная разработка и тестирование компонентов алгоритма
